Engineered a comprehensive Healthcare Fraud Audit Dashboard in Power BI, processing a $476M billing dataset to successfully isolate and analyze $248M in fraudulent claims.

Authored advanced DAX measures and applied strict filter contexts to dynamically separate legitimate claims from high-risk anomalies, ensuring accurate calculations for executive reporting.

Designed a dynamic Decomposition Tree for root-cause analysis, empowering stakeholders to trace fraud pathways from top-level categories (e.g., Phantom Billing) down to specific diagnoses and patient ages.

Implemented visual anomaly detection using custom scatter plots to instantly identify demographic billing outliers, exposing high-dollar fraudulent claims targeting specific patient age groups.

Optimized UI/UX for executive decision-making, deploying a high-contrast dark theme and standardized data formatting (millions) to increase readability and speed to insight.
